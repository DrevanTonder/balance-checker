# balance-checker
A unpaid balance checker for mining made using [Electron](https://electronjs.org) and [Vue](https://vuejs.org)

# How to Install
Clone Repository
```shell
git clone https://github.com/DrevanTonder/balance-checker.git
```
Move into Directory
```shell
cd balance-checker
```
Install npm modules
```shell
npm install
```
# Running
Run `start.bat` or `electron .`
